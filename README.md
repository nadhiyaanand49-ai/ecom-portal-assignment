# E-Commerce Portal (Assignment)

This is a simple web application built using **HTML, CSS, JavaScript**.

## Features
✔ Navigation menu  
✔ User Registration + Validation  
✔ Login Form  
✔ Orders List  
✔ Wishlist  
✔ Comments & Reviews  
✔ Customer Care  
✔ Report Issue Section  

## How to Run
Just download the project and open **index.html** in any browser.
